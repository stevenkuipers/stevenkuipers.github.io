Work

A simple guessing game. Can you guess which country belongs to a given capital? Created as an exercise in web scraping. The .js file contains a list of all countries in the world and their capitals.
A local weather single page app. It uses information from the navigator object to determine user location. Then uses this geo-data to query the open weather api. If no navigator object is available it falls back to IP address localization.
This page translates basic Latin alphabet text to Morse code.
An AJAX powered quote generator. The page fetches a new quote about design when the user requests one and offers the ability to tweet the quote.
A To-Do list powered by Jquery. All standard functionality is there. Build as a practice exercise.
First challenge on Free Code Camp: Build a tribute page. The page uses the bootstrap library for it's responsive design.
